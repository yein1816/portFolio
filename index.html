사용자 행동 기반 데이터(좋아요, 북마크, 시청 이력, 검색 기록)를 수집하여 맞춤형 추천을 제공하며, 로그인 유무에 따라 추천 방식이 다르게 동작합니다.
YouTube API를 통해 영상 데이터를 자동 수집하고, 실시간 반영 기능을 갖추고 있습니다.

  • Frontend: React, Axios, CSS
  • Backend: Spring Boot, JPA, MySQL
  • 기여 역할:
    • UI/퍼블리싱 총괄 및 화면 구현 주도 (기여도 70%)
    • 로그인 여부에 따라 영상 추천 방식 분기 처리 (전체 vs 개인화 추천)
    • 사용자 행동 기반 추천 알고리즘 구현
    • 좋아요/관심 없음(싫어요) 기능 개발
    • 좋아요한 영상 및 최근 시청 영상 조회 페이지 구현
    • 검색바 및 키워드 기반 검색 결과 페이지 개발
  • 주요 기능: 사용자 행동 기반 맞춤 영상 추천, 북마크/좋아요/싫어요, YouTube API 연동
  • API 관리: Swagger를 통한 REST API 문서 자동화 및 테스트

    • Figma 프로토타입 기반 화면 구성 주도
    • 다양한 UI 페이지 직접 제작 및 통합 (메인/상세/추천 결과 등)
